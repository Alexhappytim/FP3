## Тимошков Александр Андреевич ФП3
### ТЗ
Реализовать консольное пользовательское приложение на F# по интерполяции функций по заданным точкам.
Реализовать следующие методы интерполяции:
- Гаусс
- Лагранж
- Ньютона
- Линейная
Ввод из консоли, в формате похожем на csv, "x;y\n"
`fp3 --newton -n 4 --step 0.5`
```fsharp
< 0 0
< 1 1
< 2 2
> 3 3
> 4 4
> newton: 0 0
> newton: 0.5 0.5
> newton: 1 1
> newton: 1.5 1.5
> newton: 2 2
> newton: 2.5 2.5
> newton: 3 3
< 5 5
> newton: 3.5 3.5
> newton: 4 4
< 7 7
> newton: 4.5 4.5
> newton: 5 5
< 8 8
> newton: 5.5 5.5
> newton: 6 6
> newton: 6.5 6.5
> newton: 7 7
< EOF
> newton: 7.5
> newton: 8

```
### Реализация
Код разбит на несколько модулей
`*Interpolation.fs`
Модули под все 4 разных метода интерполяции, функция eval возвращает список посчитанных точек

`Parser.fs`
Модуль для парсинга консольного ввода и аргументов команды
`PointGenerator.fs`
Модуль для генерации промежуточных точек с шагом
`StreamProcessor.fs`
Обработка входного потока с помощью Queue

Тесты реализованны для интерполяции, парсера и работы из консоли.
